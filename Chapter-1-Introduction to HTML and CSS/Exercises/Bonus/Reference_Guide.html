HTML Reference Guide

  • HTML is the standard markup language for creating Web pages.
  • HTML stands for Hyper Text Markup Language
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

What is a HTML Tag?

  • Tags are the basic building blocks of a web page.
    They tell the browser how the information is to be displayed on the web page.
    HTML tags not case sensitive.
  • In HTML document, a tag name begins with an opening angular bracket (<) and ends with losing angular bracket (>).

    For example,
    <TagName> content </TagName>
    <TagName> - Opening Tag
    </TagName> - Closing Tag

  • The HTML tags can be categorised as:
    1. Container Tags: The tags that have both opening and closing tags are called Container Tags.
    2. Empty Tags: Empty tags contain only opening tags. They do not have a closing tag. These tags do not enclose any data.

What is a HTML Element?

An HTML element is defined by a start tag, some content, and an end tag:

Tag Description
<html> Defines an HTML document
<body> Defines the document's body
<h1> to <h6> Defines header 1 to header 6
<p> Defines a paragraph
<br> Inserts a single line break
<hr> Defines a horizontal rule
<!--> Defines a comment

HTML DOCUMENT STRUCTURE

  • The markup tags tell the Web browser how to display the web page. An HTML file is saved with. htm or .html extension.
  • The basic Structure of HTML Document is as follows:

    <html>
    <head>
    <title> Title of the web pages </title>
    </head>
    <body> Content of the page </body>
    </html>

  • A HTML page is divided into two sections called Head and Body.
  • Head contains the information which is not to be shown on the web page.
  • Body contains the information which is to be shown on the web page.

Tags in Detail

  • <HTML> Tag:
    It tells the web browser that the text contained between <html> and </html> is a web page. It can be viewed by the web browser. Every webpage coding must start with the <html> tag and end with the </html> tag.
  • <Head> Tag:
    The <Head> tag defines the header area of your web page. The information given in <Head> tag tells the computer that this information is not to be shown on the webpage. Every webpage coding must have its heading tag.
  • <Title> Tag:
    It tells the web browser that the text contained between <title> and </title> is the text that does not form part of the web page. It will be shown on the Title Bar of your web browser. The use of the title bar is not mandatory.
    If you are not using a title bar your web browser will show the file name and its path as the title of your webpage.
  • <Body> Tag:
    The <body> Tag tells the web browser that text contained between <body> and </body> tags is to be shown on the webpage. It is a container tag.
  • CENTER tag <center>:
    The <center> tag is used to put an object into the center of the web page.
  • BR (Break line) tag <br>:
    <br> tag inserts a single line break. It is useful for writing a poem or an address, where the division of lines is significant
  • HR (Horizontal Rule) tag <hr>:
    <hr> tag is used for creating a horizontal line in web page. It is used to separate content (or define a change) in web page.
  • PARAGRAPH <p> Tag:
    Normally, we create space between two paragraphs using Enter key, but in HTML browser does not understand the space created by the Enter key. Therefore we use paragraph <p> tag to inform the browser about a sentence to be started after the Enter key.
  • BOLD Tag <b>:
    Bold tag is used to display the text in bold font. This tag is generally used to give headings or to highlight important. The idea here is to use the bold tag in quick formatting situations.
  • ITALIC Tag <i>:
    The italic tags should be used to highlight a key word or phrase. These tags are not intended to stylize or shape your font face. Rather, use them to emphasize text or words.
  • UNDERLINE Tag <u>:
    Underline tag helps to underline the text.
  • SUBSCRIPT <sub> Tag:
    You can use the <sub> tag to create a subscript content (text) in the web page. This tag puts the character slightly below the line of regular text.
  • SUPERSCRIPT <sup> Tag:
    You can use the <sup> tag to create a superscript content (text) in the web page. This tag puts the character slightly above the line of regular text.

CREATING LISTS IN HTML

HTML lists appear in web browsers as bulleted lines of text. Each list type utilizes it's own unique list tag, which we'll demonstrate below. Types of lists.

  1. Ordered Lists (numbers)
  2. Unordered Lists (bullets)

Ordered and Unordered list are created using <li></li> tag.

  • Ordered List
    Ordered list <ol> is used to make the numbered sequence of items. By default, it starts with 1. Use <ol> tag to create an ordered list and <li> tag to create items of the list.

  • Unordered List
    An unordered list (<ul>) signifies to a web browser that all list items contained inside the <ul> tag should be rendered with a bullet preceding the text. The default bullet type for most web browsers is a full disc (black circle).
